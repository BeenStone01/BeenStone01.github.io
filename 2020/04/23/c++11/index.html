int arr[10];
int arr[] = {1,2,3};

array<int,10> copy = a1;
数组只能是遍历的形式来拷贝。

vector, string, array对于下标的运算要求下标必须是无符号类型，而数组则没有这个限制。

运算符重载

class Task
{
public:
    void operator()(int i)
    {
        cout << i << endl;
    }
};

int main()
{

    for (uint8_t i = 0; i < 4; i++)
    {
        Task task;
        thread t(task, i);
        t.detach(); 
    }
}
