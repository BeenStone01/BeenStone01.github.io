GPIB

Been Stone 2020-07-19 20:27:45

简介

GPIB是用来控制电子仪器的，主要用在自动化控制、测试上。标准是SCPI。

名词解释

GPIB: Gerneral Purpose Interface Bus

通用接口总线是一种设备和计算机连接的总线。大多数台式仪器是通过GPIB线以及GPIB接口与电脑相连，系统中最多不能超过15台仪器。如果是台式机或者笔记本作为主控机，需要买一个USB转GPIB的线。

VISA: Virtual Instrument Software Architecture

虚拟仪器软件结构。可以使用网线连接仪器，通过网络地址来程控仪器，也可以通过GPIB线，使用端口程控仪器。
关于visa的详细内容参见wiki

SCPI指令集: Standard Commands for Programmable Interface

可编程仪器标准命令，定义了一套用于控制可编程测试测量仪器的标准语法和命令。命令主要分为Set（控制）和Query（查询）两种。

编写程控软件的步骤：

  1. 确定编程语言。
  2. 明确手动控制仪器的流程，需要控制的仪器参数。
  3. 查找仪器命令手册（一般从仪器官网可以下载到命令手册，），查找出对应参数的控制命令。
  4. 安装GPIB或者Visa驱动（非常重要）。
  5. 调用驱动，编写自动测试流程（实现你的测试流程，做你想做的事儿）。

GPIB编程方法

  1. 直接调用NI公司提供的GPIB卡的驱动程序库来驱动GPIB卡和测试仪器进行通信（NI 公司和Keithly公司的GPIB卡可以支持这种方式）
  2. 直接调用AGILENT公司提供的GPIB卡的驱动程序库来驱动GPIB卡和测试仪器进行通信（Agilent的GPIB卡可以支持这种方式）
  3. 直接调用VISA库来驱动GPIB卡和测试仪器进行通信（NI公司和Agilent公司的GPIB卡可以支持这种方式

通过VISA库实现GPIB通信

注意事项：

  1. 必须先安装Agilent IO Library驱动，本人添加的是默认位置的库；装了驱动后有很多很好的文档 C:\Programm Files\Agilent\IO Libraries Suite\Manuals\visa.pdf
  2. 安捷伦提供SICL和VISA两种库，不推荐使用SICL，因为VISA更通用。NI也用VISA，函数名一致，都是viOpenDefaultRM,viOpen,viPrintf,viQueryf,viClose
  3. 其实不难，但要多实践，指示调用APD（头文件visa.h，库agvisa32.lib），APD常用的就是几个，viOpenDefaultRM,viOpen,viPrintf,viQueryf,viClose…还有viSetAttribute,viScanf等，其他的主要是SCPI命令，大多是标准化的，查查编程文档即可。
viSetAttribute(vi,VI_ATTR_TMO_VALUE,20000); //超时时间s
  1. 较新的技术是：LXI（LAN Extention Interface），网口控制。因为一方面是GPIB接口较贵（3k-5k），设备也只能接十几个（通常足够了）；
  2. 扫描时需要单次同步，这样才准确
  3. 应该多在Agilent和NI官网看看
  4. 好书：《NI-VISA Programmers Man》，《Agilent VISA User’s Guide》
  5. 自动搜索仪器，根据*IDN知道什么仪器

VISA API

ViStatus status; 
ViSession defaultRM;

status = viOpenDefaultRM(&defaultRM);    //打开缺省资源管理器资源对话通道

用于初始化并定位VISA系统。其中，defaultRM表示一个VISA库函数中专用的与仪器之间进行通讯的ViSession类型的对象，在调用VISA库时，这是第一条必须使用的语句，表示VISA库数已经启动。

viOpen(sesn,rsrcName,accessMode,timeout,vi);    //打开特定资源的对话通道
ViSession Dev1;
viOpen(defaultRM,pAddress,VI_NULL,VI_NULL,&Dev1);

用于建立计算机与仪器之间联系的会话。
本函数紧跟在viOpenDefaultRM(&defaultRM)函数后，作为第二条命令使用，共有五个参数。 sesn
表示指向第一条语句中ViSession对象的指针； rsrcName
表示要控制仪器的参数值，包括其总线结构样式、地址，使VISA库能定位此设备； accessMode
表示VISA系统对仪器的访问模式，一般都直接取默认值，表示对设备完全访问。 timeout
表示超时时间，单位为ms,一般取默认值，表示不设超时控制。 vi
是一个指针，指向与被控设备相关的ViSession对象，访问它可随时获取设备任何时刻的被程控状态。

viPrintf(Dev,writeFmt params);  //按设定格式将数据传送到器件中

用于发送设备控制指令，设置其初始参数值或指示其进行某项操作，使设备动作或查询其状态。其中Dev表示与当前被控设备先关的ViSession对象；writeFmt params是字符串参数，表示控制指令，是符合SCPI的。

代码样例


#include "visa.h"
#pragma comment(lib, "VISA32.lib")

int main(void)
{

    ViStatus  status;
    ViSession rsrcManager, instHandle;

    /* Open Default RM */
    status = viOpenDefaultRM(&rsrcManager);
    if (status < VI_SUCCESS) 
    {
        /* Error Initializing VISA...exiting */
        return -1;
    }

    /* Access other resources */
    status = viOpen(rsrcManager, "GPIB::1::INSTR", VI_NULL, VI_NULL, &instHandle);
    if(status != VI_SUCCESS)
    {
        return -1;
    }
    else
    {
        viGetAttribute(instHandle, VI_ATTR_MODEL_CODE, &moduleID);
        viSetAttribute(instHandle, VI_ATTR_TMO_VALUE, 2000);
    }
    /* Use device and eventually close it. */
    viClose(instHandle);
    viClose(rsrcManager);
    return 0; 
}


BOOL CXXXDlg::InitPower()   
{  
    char chStatusDesc[_MAX_DIR] = {0};  
    int retCnt=0;     
    char instrDesc[100]={0};      
    ViFindList find_list;     
    float voltSetting, currSetting;  
    voltSetting = atof(ini_powervolt);  
    currSetting = atof(ini_powercurrent);  
    VISAstatus=viOpenDefaultRM(&defrm);  
    if (VISAstatus != VI_SUCCESS)  
    {  
        return FALSE;  
    }  
    VISAstatus = viFindRsrc(defrm,"GPIB?*INSTR",&find_list,(ViPUInt32)&retCnt,instrDesc);  

    VISAstatus=viOpen(defrm,instrDesc, VI_NULL, VI_NULL, &session);  

    if (VISAstatus!=VI_SUCCESS)  
    {  
        return FALSE;  
    }  
    //Set voltage  
    viPrintf(session,"VOLT %f \n",voltSetting);     
    //Set current level  
    viPrintf(session,"CURR %f \n",currSetting);  
    return TRUE;   
}

VISA操作流程

graph TD 
    A(开始)-->B[打开资源管理器:viOpenDefaultRM]
    B-->C[打开特定资源:viOpen]
    C-->D[读写设备数据:viRead/viWrite]
    D-->E[关闭特定资源:viClose]
    E-->F[关闭资源管理器:viClose]
    F-->G(结束)

VisaGpib使用流程


graph TD

    A(开始)-->B[Gpib_Flag初始化为true]
    B-->C[打开资源管理器:OpenRM]
    C-->D[查询设备:FinRsrc]
    C-->E[获取设备名字:GetInstrurmnt_Name]
    C-->F[打开设备:OpenDev]
    F-->G[发送指令:Send]
    F-->H[读取信息:Read]
    F-->I[查询信息:Query]
    G-->J[关闭设备:CloseDev]
    H-->J[关闭设备:CloseDev]
    I-->J[关闭设备:CloseDev]
    J-->K[关闭资源管理器:CloseRM]
    K-->L(结束)
